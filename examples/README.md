# BAEX Framework Examples

This directory contains demonstrations, primitive showcases, and API stress-tests for the BAEX framework.

## Structure

Each example should reside in its own subdirectory:

```text
examples/
└── example-name/
    ├── index.html
    ├── main.ts
    └── style.css
```

## Creating a New Example

1. Create a new folder under `examples/`.
2. Implement your example using standard BAEX patterns (`import { BAEX } from '../../src/core/baex';`).
3. Ensure your example can be bundled (or referenced) by the project's build system.
